v1.4.0 April 12, 2026
  • Search icon in compact mode — click to expand and search through your clips
  • Fixed item pin button not responding to clicks
  • Fixed live checkout URL for license purchases
v1.3.0 April 3, 2026
  • Multi-file groups — drop multiple files into Klipt and drag them all out together
  • Lock to screen — lock icon keeps the panel visible while you work in other apps
  • Item pin moved to preview — pin button now on the top-right of each item
  • Auto-tab switching — opening Klipt defaults to the tab matching your last copied item
  • Movable preview panel — drag the Space-key preview anywhere on screen
  • Fixed duplicate screenshot entries from macOS temp files
  • Fixed panel appearing on wrong Space — now always moves to the active Space
  • Disabled macOS Automatic Termination to prevent silent app suspension
  • Screenshot flash extended to 3 seconds
  • All mouse interactions now work — fixed drop overlay blocking button clicks
v1.2.0 March 13, 2026
  • Images now stored as files on disk in ~/Pictures/Klipt/ instead of inline data
  • Show in Finder for screenshots and images via right-click context menu
  • Save Image to… option for saving screenshots to a custom location
  • Show Klipt menu bar item to open the panel without the hotkey
  • Right-click context menu now works on all items (files, images, text)
  • Improved paste reliability with global HID event tap
  • Security hardened: license keys stored in Keychain, file permissions restricted
  • Prevented App Nap from disabling hotkeys after system idle
  • Cross-app image drag with PNG, TIFF, and file URL support
  • Capped clipboard history to 100 items per category
  • Fixed panel becoming unresponsive after prolonged use
  • Install instructions added to the website
v1.1.0 March 9, 2026
  • File previews — thumbnails for images, PDFs, documents, and more via QuickLook
  • Dynamic panel height — window resizes to fit image and file previews
  • Auto-switch to latest item when a new clip is added while panel is open
  • Menu bar menu with Settings and Quit options
  • Stable window positioning — panel remembers its position across sessions
  • Fixed false drag triggers when clicking browser tabs
  • Fixed window drifting to corner after expand/collapse
